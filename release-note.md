# Release Note - Kenpix

## Version 1.0.0

Kenpix versi 1.0.0 adalah rilis awal aplikasi desktop untuk konversi gambar yang fokus pada kemudahan penggunaan, performa ringan, dan tampilan modern.

## Fitur Utama

- Drag and drop gambar untuk proses konversi yang lebih cepat
- Dukungan format input: PNG, JPG, JPEG, WEBP, dan AVIF
- Dukungan format output: PNG, JPG, JPEG, WEBP, dan AVIF
- Pemilihan folder penyimpanan hasil konversi
- Progress bar saat proses konversi berjalan
- Tombol cancel untuk menghentikan proses konversi
- Halaman settings untuk mengelola preferensi aplikasi
- Riwayat konversi yang tersimpan secara lokal
- Tombol untuk membuka folder hasil langsung dari history
- Tampilan modern dengan dark theme dan glassmorphism

## Peningkatan pada Versi Ini

- Aplikasi dibuat dengan arsitektur desktop menggunakan Tauri agar tetap ringan
- Navigasi dipisahkan menjadi Home, Settings, dan History untuk pengalaman yang lebih terstruktur
- Informasi hasil konversi disimpan agar pengguna dapat melacak file yang sudah diproses
- Desain antarmuka dibuat lebih rapi dan responsif untuk desktop

## Catatan Rilis

- Riwayat konversi disimpan secara lokal di perangkat pengguna
- Lokasi output default dapat diubah melalui menu Settings
- File hasil konversi akan disimpan ke subfolder berdasarkan format yang dipilih

## Rencana Pengembangan Berikutnya

- Penambahan lebih banyak opsi pengaturan
- Optimasi manajemen file hasil konversi
- Peningkatan UX untuk proses batch conversion
- Penambahan fitur preview atau detail metadata gambar
